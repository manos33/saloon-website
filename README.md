## 🎨 CUSTOMIZATION GUIDE

### Change Colors:
Open `index.html` and find the `:root` section (around line 14):
```css
:root {
    --wood-brown: #8b5e3c;
    --warm-cream: #f5ebdd;
    --soft-white: #fff9f0;
    --terracotta: #d67c5e;
    --sage-green: #7a8450;
}
```
Change any hex code to adjust colors site-wide.

### Change Photos:
Replace files in `images/` folder:
- `hero.jpg` - Main background (1920x1080 recommended)
- `about.jpg` - About section (800x600 recommended)
- `gallery1-6.jpg` - Gallery photos (800x800 recommended)

### Update Text:
Search in `index.html` for any text you want to change.
All content is clearly labeled with comments.

## 🆘 TROUBLESHOOTING

**Website won't load:**
- Clear browser cache
- Try different browser
- Check file paths in images

**Photos not showing:**
- Verify images are in `images/` folder
- Check file names match exactly
- Ensure images are .jpg format

**Form not working:**
- Check FORM-SETUP.md
- Verify service is connected
- Test internet connection

**Mobile menu not opening:**
- Clear cache
- Check JavaScript is enabled
- Try different device
